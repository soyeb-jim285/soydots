## App Launcher — prefix modes (Super+Space)

_Type a prefix then a space to switch modes. Shown as chips in the launcher; the empty launcher also suggests them._

### Prefixes

| Keys | Action |
| --- | --- |
| `(type)` | Apps + universal search — default, no prefix |
| `k <query>` | Keybindings search (Enter copies the chord) |
| `/ <expr>` | Calculator |
| `: <query>` | Emoji picker |
| `cb <query>` | Clipboard history |
| `p <query>` | Packages — pacman / AUR |
| `i <query>` | Flatpak — flathub |
| `' <query>` | File search |
| `> <cmd>` | Shell runner |
| `g <query>` | Web search — engine keyword (g, gh, aw…) |

## tmux — prefix is Ctrl+Space

_'prefix X' = press Ctrl+Space then X. Capital resize binds are repeatable. Copy mode uses vi keys._

### Panes

| Keys | Action |
| --- | --- |
| `prefix | / -` | Split horizontal / vertical (cwd) |
| `prefix h/j/k/l` | Select pane left/down/up/right |
| `Alt+Left/Right/Up/Down` | Select pane (no prefix) |
| `prefix H/J/K/L` | Resize pane (repeatable) |
| `prefix m` | Toggle pane zoom |
| `prefix > / <` | Swap pane next / prev |
| `prefix b` | Break pane to own window |
| `prefix x` | Kill pane |

### Windows

| Keys | Action |
| --- | --- |
| `prefix c` | New window (cwd) |
| `prefix X` | Kill window |
| `prefix Space` | Last window |
| `Alt+h / Alt+l` | Prev / next window (no prefix) |
| `Alt+1..9` | Jump to window 1–9 (no prefix) |

### Copy mode

| Keys | Action |
| --- | --- |
| `prefix v` | Enter copy mode |
| `v` | Begin selection |
| `y` | Copy to Wayland clipboard + exit |
| `Ctrl+v` | Toggle block selection |
| `Escape` | Cancel copy mode |
| `prefix p / P` | Paste / choose buffer |

### Popups

| Keys | Action |
| --- | --- |
| `prefix w` | Session list |
| `prefix T` | Scratch pad |
| `prefix g` | LazyGit (cwd) |
| `prefix S` | SSH menu |

### Misc

| Keys | Action |
| --- | --- |
| `prefix r` | Reload config |

## Neovim — LazyVim (leader = Space)

_Stock LazyVim defaults (no custom user keymaps). <leader> = Space. See :help LazyVim or <leader>sk to search keymaps live._

| Keys | Action |
| --- | --- |
| `Ctrl+h/j/k/l` | Move to window left/down/up/right |
| `Ctrl+arrows` | Resize window |
| `Shift+H / Shift+L` | Prev / next buffer |
| `<leader>bd` | Delete buffer |
| `<leader>-  /  <leader>|` | Split below / right |
